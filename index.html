About MakeItSo

MakeItSo is a bare startup theme with the minimum of styling. It is set up with the assumption a website is centered and full width. It is however very easy to create a max-width and centered layout like this one.

This is not a HTML framework

MakeItSo strives to not give you unneeded and excessive markup, styles and classes or pre-made design patterns. You don't need Foundation or Bootstrap to built a sleek and efficient front-end, nor should you need MakeItSo for that matter. For protoyping, a framework like Foundation can be handy.

  • Normalize.css is used as default to align browsers' rendering appearrance and positioning. The rest is bare bones...
  • Need a grid in place at a certain break-point? Just declare (wrapper) widths in percentages, float them and be done with it. Not 'col-6', 'spacer-3' or 'med-12-4-1-grid_home_bloat' classes... See the example classes in _theme.scss for 2 or three columns.

Use your head, your creative skills and ye'olde CSS plus good accessible HTML!

HTML elements example page

The article example page contains base HTML elements to show how the base styling looks in this setup. If you still miss an element here for reference, please let me know. The form example page shows you the default generic styling of a form and form elements. Go check them out.

MakeItSo is responsive and built mobile first. Any changes and additions you make should be added in a mobile first workflow. As the viewport widens with every larger sized medium, styles get applied and/or overwritten.
